# Базовый Backend. Модуль 14. Сессии и Cookie.

Практическая работа

## Задание 14.8

- Разработана главная страница демо-версии сайта для SPA-салона.
- Размещены фото, услуги, акции салона.
- Реализована возможность регистрации на сайте (для регистрации необходимо ввести email, пароль, имя и дату рождения). 
- Реализована возможность входа зарегистрированного пользователя в личный кабинет (для входа необходимо ввести указанные при регистрации email и пароль).
- В шапке страницы личного кабинета отображается имя вошедшего пользователя и количество дней до его дня рождения.
- При регистрации нового пользователя ему предлагается индивидуальная акция с таймером      обратного отсчета до окончания акции.
- Реализовано поздравление пользователя с днем рождения и предлагается скидка по этому  случаю.
- Реализована возможность выхода из личного кабинета.
- Реализованы все функции по условию задания.
- Хранение данных пользователей реализовано в файле в json формате, пароль хранится в зашифрованном виде.

- текущий зарегистрированный пользователь: login - email@example.com, pass - 123456
